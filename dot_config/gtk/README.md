# GTK Configuration

GTK 3/4 theme settings.

## Files

- `gtk.css` - Custom GTK styling
- `colors.css` - Color scheme (generated by matugen)
- `settings.ini` - GTK settings

## Features

- Material You color scheme integration
- Custom widget styling
- Consistent theming across GTK apps

## Theme

Uses `adw-gtk3-dark` base theme with Material You colors overlaid via `colors.css`.
