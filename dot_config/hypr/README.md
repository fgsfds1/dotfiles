# Hyprland Configuration

Wayland compositor configuration with lock screen and idle management.

## Files

- `hyprland.conf` - Main compositor config (keybinds, window rules, animations, env vars)
- `hyprlock.conf` - Lock screen appearance and behavior
- `hypridle.conf` - Idle management (auto-lock, screen dimming)
- `wallpaper_switcher.sh` - Automatic wallpaper rotator (30-minute intervals)
- `colors.conf` - Color scheme (generated by matugen)

## Features

- Custom keybinds for window management and app launching
- Workspace rules and window positioning
- Animations and visual effects
- Environment variables for Wayland/XDG/Qt/GTK
- Auto-lock after idle timeout
- Screen dimming before lock
- Material You color scheme integration

## Key Applications

Defined via environment variables in `hyprland.conf`:
- Terminal: `kitty`
- File Manager: `nautilus`
- Launcher: `rofi`
- Browser: `firefox-developer-edition`
- Locker: `hyprlock`
- Editor: `nvim`

## Reload

After editing configs:
```bash
hyprctl reload
```

Or it reloads automatically via chezmoi post-apply hook.
