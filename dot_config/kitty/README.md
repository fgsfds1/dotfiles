# Kitty Configuration

GPU-accelerated terminal emulator.

## Files

- `kitty.conf` - Terminal settings (font, cursor, performance)
- `colors.conf` - Color scheme (generated by matugen)

## Features

- GPU acceleration
- Font ligatures support
- Material You color scheme
- Customizable cursor
- Performance optimizations

## Reload

Kitty reloads config automatically on changes.
