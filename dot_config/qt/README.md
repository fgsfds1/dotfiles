# Qt Configuration

Qt5ct theme settings for Qt applications.

## Files

- `qt5ct.conf` - Qt configuration
- `colors/` - Color scheme (generated by matugen)

## Features

- Material You color scheme integration
- Consistent theming with GTK apps
- Qt application styling via qt5ct

## Usage

Requires `QT_QPA_PLATFORMTHEME=qt5ct` environment variable (set in Hyprland config).
