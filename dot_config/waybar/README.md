# Waybar Configuration

Status bar for Hyprland showing system info and workspaces.

## Files

- `config.json` - Bar layout and module configuration
- `style.css` - Bar styling (fonts, spacing, borders)
- `colors.css` - Color scheme (generated by matugen)

## Modules

### Left
- Hyprland workspaces
- Active window title

### Right
- CPU usage and frequency
- Memory usage
- Temperature (CPU via sensors, GPU via nvidia-smi)
- Volume (PulseAudio)
- Network status
- Battery
- Package updates (pacman)
- Keyboard layout
- System tray
- Clock

## Interactions

- **Volume**: Click to open `pwvucontrol`
- **Network**: Click to open `nm-connection-editor`

## Reload

```bash
killall -SIGUSR1 waybar
```

Or it reloads automatically via chezmoi post-apply hook.
